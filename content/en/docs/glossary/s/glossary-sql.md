  **S**tructured **Q**uery **L**anguage (SQL).
  The language used in the MySQL database to query and manipulate the data.
---

SQL is used by Stroom's database migration scripts to modify the structure of the database schema and the data contained within it.
Sometimes it is necessary for Stroom administrators to manually run SQL statements/queries as part of pre-migration checks.
